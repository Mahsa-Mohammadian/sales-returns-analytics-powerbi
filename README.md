# Sales & Returns Analytics Dashboard
![Sales & Returns Analytics Dashboard](Sales_Analytics_Dashboard_PowerBI.png)
This project analyzes sales performance and return behavior using retail transaction data.

## Tools Used
- PostgreSQL
- Power Query (Excel)
- Power BI

## Project Overview
The goal of this project was to understand revenue trends, identify top-performing items, and analyze return patterns.

## Key Insights
- Revenue peaks in Q4, showing strong year-end growth
- A small number of items generate a large share of revenue
- Return values are concentrated in a limited number of items
- Some high return values are driven by operational charges such as shipping and service fees

## Dashboard
A Power BI dashboard was created to visualize trends, top products, and returns.
